# 演習②：コード補完を体験する

## 目的

コメントや関数名から、
GitHub Copilot がコードを提案することを体験する。

## 手順

1. exercise02/utils.py を開く
2. 以下を入力する

```python
# 文字列が回文か判定する（大文字小文字は無視、記号と空白も無視）
def is_palindrome(text):
```

3. Enterキーを押す
4. Copilotの提案（灰色文字）を確認する
5. Tabキーで採用する
6. Escキーで却下する

## 確認ポイント

* Copilotがコードを提案したか
* コメントの内容を理解しているか
* 回文判定のロジックになっているか

## 学び

コメントもプロンプトの一種である。

コメントが具体的になるほど、
Copilotの提案も具体的になる。

例

悪い例

```python
# 回文判定
def is_palindrome(text):
```

良い例

```python
# 文字列が回文か判定する（大文字小文字は無視、記号と空白も無視）
def is_palindrome(text):
```

AIコーディングでは、
コードだけでなくコメントの書き方も重要である。

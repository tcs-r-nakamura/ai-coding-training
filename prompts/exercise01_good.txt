AIコーディング研修で使用する「タスク管理ダッシュボード」を作成してください。

# 背景

受講者はHTML/CSS/JavaScript初心者です。

このアプリは教材として使用するため、以下を学べる構成にしてください。

- 可読性
- 保守性
- 責務分離

# ゴール

ブラウザで開くだけで利用できるタスク管理ダッシュボードを作成してください。

単なるTODOリストではなく、

「現在のタスク状況を一目で把握できる管理画面」

を目指してください。

# 必須機能

## タスク管理

- タスク追加
- タスク編集
- タスク削除
- タスク完了
- タスク未完了への戻し

## タスク情報

各タスクは以下の情報を持つこと。

- タイトル
- 説明
- 作成日
- 期限日
- ステータス

# ダッシュボード表示

以下をカード形式で表示してください。

- 総タスク数
- 完了タスク数
- 未完了タスク数
- 期限切れタスク数

# 集計表示

以下を表示してください。

- 今日のタスク数
- 今週のタスク数
- 進捗率

# グラフ

Canvas APIを使用し、以下の割合を円グラフまたは棒グラフで表示してください。

- 完了タスク
- 未完了タスク

※ 外部ライブラリは禁止です。

# フィルター

以下のフィルターを用意してください。

- すべて
- 未完了
- 完了済み
- 期限切れ

# 一括操作

以下の一括操作を用意してください。

- すべて完了
- 完了済みタスク削除

# データ保存

localStorageを使用してください。

画面上部にアプリタイトルを表示してください。

- 総タスク数
- 完了数
- 未完了数
- 期限切れ数

中央エリアに以下を表示してください。

- 進捗率
- 完了率グラフ

その下に以下を表示してください。

- 今日のタスク
- 今週のタスク
- 期限切れタスク
- タスク一覧

# デザイン要件

- モダンな管理画面風
- カードレイアウト
- 適切な余白
- 見やすい文字サイズ
- レスポンシブ対応
- スマホ表示対応
- 初心者にも分かりやすいUI

# 制約条件

- HTML
- CSS
- JavaScript

のみ使用してください。

以下は禁止です。

- 外部ライブラリ
- CDN
- フレームワーク
- サーバー

index.html を開くだけで動作すること。

# コーディングルール

- HTML/CSS/JavaScriptを分離する
- 関数ごとに責務を分離する
- 分かりやすい変数名・関数名を使う
- 日本語コメントを付与する
- 重複コードを避ける
- 初心者が理解できるコードにする
- 1つの関数に複数の責務を持たせない

# 出力内容

以下を必ず出力してください。

1. フォルダ構成
2. 各ファイルのコード
3. 起動方法
4. 操作方法
5. 実装機能一覧
6. 初心者向けコード解説
